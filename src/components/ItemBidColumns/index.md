作为投标的入口。

## API

| 参数      | 说明                                      | 类型         | 默认值 |
|----------|------------------------------------------|-------------|-------|
| links | 路由地址 | String | - |
| cover | 封面图 | String | - |
| title | 标题 | String | - |
| description | 内容描述    | ReactNode  | -    |
| annual | 年利率 | Number | - |
| month | 投资期限 | Number | - |
| num1 | 已融资金额 | Number | - |
| num2 | 需融资金额 | Number | - |
| date | 发标日期 | String | - |
